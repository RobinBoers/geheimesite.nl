About the site

My site is built using beautiful, readable, handwritten HTML and some ~200 lines of handcrafted CSS. It's hosted at bHosted. All content on it—everything authored by me—is in the public domain (specifically CC0). The source code is also publicly available and licensed AGPLv3.

The server has been configured to compress and sent the correct caching headers. It also enforces a very strict CSP, which disables any scripts and insecure content. I also send a bunch of privacy-related headers, because the modern web absolutely sucks by default, and I prefer visitors to not be tracked by their own browsers.
