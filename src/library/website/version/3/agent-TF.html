Over agent TF

deze gozer is de beste spion ever. agent TF is beheerder van het camerasysteem van de DDDD. hij is ook de leider van de geheim-agent-school!

hij train geheim agenten van een kind tot een spion die daarna bij de GAD (Geheim Agent Dienst) kan werken!

ook heeft hij een zweefschild en controle over het camera systeem van de DDDD... oeps dat had ik al verteld!
